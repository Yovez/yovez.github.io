About Me.

I'm an experienced Java developer with over 4 years experience primarily creating Minecraft server plugins. I grew bored of Minecraft server plugin development due to the repition of my work, and the loss of enjoyment for the game and the community. The next big step was web development and I'm enjoying it. I've been relearning HTML, CSS, and JS with small personal projects here and there. Online tutorials have been a big help, but working with the code hands on through my own ideas and plans is where I strive to learn new material.

I am very interested in becoming a full-stack developer, my primary skill set includes JavaScript, React, Node.js, Express, Mongoose, PugJS. I am trying to learn those skills to an intermediary level at the least. I also plan to learn Java Spring, Vue, and Angular in the near future.
